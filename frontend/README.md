# Khejur Ghee Semai - E-commerce Website

## Project Overview
A complete e-commerce website for "Khejur Ghee Semai" - a traditional Bengali sweet product. The website is built with HTML, Tailwind CSS, and JavaScript.

## Features
- Responsive design that works on all devices
- Home page with slider and featured products
- Shop page with filtering options
- Product details page with reviews
- Shopping cart functionality
- Checkout process
- User account management
- Information pages (About, Contact)

## Technologies Used
- HTML5
- Tailwind CSS for styling
- Vanilla JavaScript for interactivity
- Font Awesome for icons
- Local Storage for cart management

## Pages Included
1. Home (index.html)
2. Shop (shop.html)
3. Product Details (product.html)
4. Cart (cart.html)
5. Checkout (checkout.html)
6. Account (account.html)
7. About Us (about.html)
8. Contact (contact.html)

## How to Run
1. Download all files maintaining the folder structure
2. Open index.html in any modern web browser
3. Navigate through the site using the menu
